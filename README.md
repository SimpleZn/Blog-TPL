###  Usage

1. Init

```
git clone https://github.com/SimpleZn/Blog-TPL.git
cd Blog-TPL
npm install
```

2. Modify

Modify _config.yml file with your own info.

3. Start / Build

```
hexo serve // run hexo in local server
hexo clean && hexo generate // build out the public static files.
```
